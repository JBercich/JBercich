### Welcome to my page (はじめまして) ! 🌸

I'm a Data Scientist 👨‍🔬 with @pendingai helping to leverage data for intelligent AI-driven drug discovery. I have recently graduated from the University of Sydney 🇦🇺 and Kyoto University 🇯🇵 where I studied a double degree in Advanced Computing (Honours) and Data Science. 

I specialise in all things AI from big data ETL to implementing machine learning experiment pipelines. Keeping up-to-date with researching cutting-edge SOTA models, I enjoy diversifying my skills across the many subfields of the deep learning community. It doesn't end there though, my passion for efficient algorithm design, optimisation and (really just) all things software has me exploring all the different areas of programming. But my real goal is to use these technologies to _help solve deep tech issues through innovation and sustainable methods_.

- 🔭 I’m currently working on _many things_ (I can't keep track... 😓)
  
- 🌱 I’m currently learning about Graph Neural Networks 🕸, Quantum Markov networks and game development 🕹 paradigms
